## How to download the project


```bash

  # Clonar o repositório
  $ git clone https://github.com/DevTeles/goodplate-ia

  # Entrar no diretório
  $ cd goodplate-ia

  # Instalar as dependências
  $ yarn

  # Iniciar o projeto
  $ yarn start

  # De acordo com sistema operacional
  $ yarn android
  # ou
  $ yarn ios
```
